# Question 5: Trigonometry (Finding Hypotenuse)

**Question ID:** Mixed-5  
**Source:** Exercise Mixed - Question 5 - Page 999  
**Question Number:** 5/20 (Revision 2)

### Question Text
Franziska has a right triangle with a 53° angle and an adjacent side of 8 cm. Find the hypotenuse to 1 decimal place.

### Result
✅ **OBJECTION SUSTAINED (Correct)**

**My Answer:** 13.3  
**Correct Answer:** 13.3 cm

### Solution Verification
Using cosine: cos(θ) = adjacent / hypotenuse
- cos(53°) = 8 / h
- h = 8 / cos(53°)
- h = 8 / 0.6018 ≈ 13.3 cm ✓
