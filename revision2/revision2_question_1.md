# Question 1: Pythagorean Theorem (3-4-5 Triple)

**Question ID:** Mixed-1  
**Source:** Exercise Mixed - Question 1 - Page 999  
**Question Number:** 1/20 (Revision 2)

### Question Text
Phoenix examines a right triangle with legs 9 cm and 12 cm. Find the hypotenuse.

### Result
✅ **OBJECTION SUSTAINED (Correct)**

**My Answer:** 15  
**Correct Answer:** 15 cm

### Solution Verification
Using Pythagorean theorem: c² = a² + b²
- c² = 9² + 12²
- c² = 81 + 144 = 225
- c = √225 = 15 cm ✓

This is the 3-4-5 Pythagorean triple scaled by 3.
