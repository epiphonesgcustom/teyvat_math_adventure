**Source:** Exercise Mixed - Question 4 - Page 999

### Question Text
Apollo solves x² - 5x + 6 = 0 by factoring. Find the smaller value of x.

### Result
✅ **OBJECTION SUSTAINED (Correct)**

**My Answer:** 2  
**Correct Answer:** 2

### Solution Verification
Factoring: x² - 5x + 6 = (x - 2)(x - 3) = 0
- x - 2 = 0 → x = 2
- x - 3 = 0 → x = 3
- Smaller value = 2 ✓
